# Taumasyang Cask

一些额外的 Homebrew 应用配方。

Additional Homebrew casks.

## 如何安装？How do I install these casks?

```sh
brew install taumasyang/cask/<cask>
```

On March 31, 2025, WeChat launched the unified desktop version 4.x. Due to delays in updating the official Homebrew tap and some features of the new version not meeting expectations, both the new and old version recipes have been temporarily included. They may be removed in the future depending on circumstances.
